== Installation

If you want to use this extension, you need to add the `io.quarkiverse.twitter4j:quarkus-twitter4j` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.twitter4j</groupId>
    <artifactId>quarkus-twitter4j</artifactId>
</dependency>
----
